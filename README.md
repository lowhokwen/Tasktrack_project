TaskMaster – Productivity Mobile App

TaskMaster is a cross-platform productivity application built using Ionic, Angular, Firebase, and Capacitor. It provides task management, calendar scheduling, a Pomodoro timer, analytics, native device integrations, and a simulated monetization system.

1. Features Implemented

✔ Task Management
- Add tasks
- Toggle complete
- Delete tasks
- Task detail view
- Tasks stored per user in Firebase

✔ User Authentication
- Register
- Login
- Logout
- Firebase secured rules

✔ Calendar Page
- Monthly calendar
- Highlights days with tasks
- View tasks for selected date
- Local notifications
- Share API

✔ Pomodoro Timer
- 25 min countdown
- Start / Pause / Reset
- Lofi music toggle

✔ Analytics (Pro Feature)
- Completion rate pie chart
- Monthly tasks bar chart
- Locked for free users
- Unlock via simulated Pro purchase

2. Native Device Features
- Local Notifications
- Share API

3. Monetization
- Simulated “Pro” version using Ionic Storage
- Upgrade / Restore / Reset

4. How to Run
npm install
npm start

5. Technology Stack
- Ionic 7
- Angular 17
- Firebase Authentication
- Firestore
- Capacitor Plugins
